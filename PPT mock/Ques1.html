Explain all the CSS positions(static, fixed, sticky, relative, absolute) with one code example each

Static Position :-

it's a default position for Html.it normally flows with the web page.

Fixed Positon:-

By the itself it says that its position is fixed or stays in same given position even if page is scrolled.

Sticky Position :-

When posiion Sticky is given to any of the object it floats with web page and stick or we can say fixed at a given position

Relative Position:-

When Relative position is given to any of the object it does'nt leave its position but move according to its own normal position,here parent position is its own position

Absolute Position:-

In this object move according to its parent position. here object leave its original position and move to a given position
