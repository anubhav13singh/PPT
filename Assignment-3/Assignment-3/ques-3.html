Explain the concept of Breakpoints in Responsive Web Design and How They are used in Media Queries.

Breakpoints simply means the screen width or in simple words pixel value.when a developer design a web page they give a certain value in pixel to give different design according to screen width and to make a web page full responsive


ex- @media( min-width:300px)and (max-width:600px){ color: blue; fontsize: 20px; } in this whenever the screen size is in between 300px to 600px color and font size will get applied.
