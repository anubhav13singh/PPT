What are the Different Types of Selectors in CSS & what are the advantages of them?

Css Selector selects the HTML elements for styling purpose.There are 7 types of Selector

  • Element selector

    • it selects element based on element tags ex- p,h1, span, div etc..

  • ID selector

    • it selects elements with specific class name given to element. it is denoted by # symbol

  • class selector

    • it selects elements with specific class name given to element. it is denoted by . symbol

  • Universal selector

    • it is used to select all the element in HTML document. it is denoted by * symbol

  • Attribute selector

    • it selects elements with specific Attribute of an element. ex-[href] { background-color: lightgreen; color: black; font-family: monospace; font-size: 1rem; }

  • Pseudo-class selector

    • it selects element that are in specific state. it starts with : and followed by prooerties.ex- :hover, :focus, :link, :visited

  • pseudo-element selector

    • it act as a whole new element is added.it starts with :: followed by keyword. it gives style the specific part of an element. ex:- ::first-line can be used to change the font of the first line of a paragraph.
