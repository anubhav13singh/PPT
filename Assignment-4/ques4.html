What are the major features introduced in ECMAScript 6?

ES6 comes with significant changes to the JavaScript language. It brought several new features like, let and const keyword, rest and spread operators, template literals, classes, modules ETC..

we can use in your everyday JavaScript coding are :-

        let and const Keywords
        Multi-line Strings
        Default Parameters
        Template Literals
        Destructuring Assignment
        Enhanced Object Literals
        Promises
        Classes
        Modules
        Arrow Functions
